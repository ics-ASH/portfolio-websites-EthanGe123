A computer is an electronic device used to process and store data, consisting of hardware (such as CPU, memory, storage devices) and software (such as operating system and application programs).

The MacBook is a high-performance personal computer manufactured by Apple that runs the macOS operating system.

PCs are computers for personal use that include various makes and models and typically run Windows or Linux operating systems.

The Raspberry Pi is a small single-board computer used in education and embedded systems that runs a Linux-based operating system.

Week #2 - Computer Hardware

Computer hardware comprises physical components like the CPU, memory, and storage devices, essential for processing and storing data. The motherboard acts as the main circuit board connecting various components. Input devices, such as keyboards and mice, enable user interaction, while output devices, like monitors and printers, display processed information. Graphics Processing Units (GPUs) handle graphical tasks, and networking components allow connection to networks. Cooling systems prevent overheating, and understanding hardware components is crucial for building and maintaining computer systems.
